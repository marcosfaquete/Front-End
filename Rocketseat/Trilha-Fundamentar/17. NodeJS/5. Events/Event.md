

# Event Module 

    É u mecanismo para
    [] Disparar eventos
    [] Ouvir eventos
    [] Fazer alguma ação quando ouvir o evento
    [] É a bease para muito outros módulos como: http, stram, file system, etc..


# O que vamos aprender?

    [] Utilizar os eventos
    [] Disparar eventos
    [] Ouvir eventos
    [] executar ações para determinados eventos
