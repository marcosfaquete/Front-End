8. Tags de divisão e containers

Essas tags ajudam a estruturar o layout da página.

(div) Um bloco genérico usado para agrupar outros elementos. Muito usado em conjunto com CSS para criar layouts.

(span) Um container inline (dentro de uma linha) usado para estilizar ou agrupar pequenos trechos de texto.

10. Tags semânticas (HTML5)

Introduzidas no HTML5, essas tags melhoram a organização e a acessibilidade do código. (header): Define o cabeçalho de uma página ou seção.
(footer): Define o rodapé.
(main): Representa o conteúdo principal da página.
(article): Representa um artigo independente (como posts de blog).
(section): Define uma seção temática da página.
(aside): Usado para conteúdo relacionado, como barras laterais.
(nav): Define a navegação (menus, links).

13. Tags para interatividade

Estas tags permitem adicionar mais interação e dinamismo à sua página.
(details): Cria uma seção que o usuário pode expandir/recolher.
(summary): Define o título do (details).
(dialog): Cria uma caixa de diálogo (como um modal).
